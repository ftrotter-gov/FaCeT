Medicare Provider Types
=================

PECOS gives rise to three separate classes of identifiers, each with a different prefix. The prefix determines whether the identifier is for a practitioner, a DME supplier, or for ordering and referring—these are fundamentally the three core relationship types within the Medicare PECOS system.

However, the second number follows a repeating pattern. With a few notable exceptions, such as 70 and some others, these are effectively identical across the three classes. Because of that, we can extract just those second digits and treat them as the Medicare specialty code.

The Medicare specialty code is therefore the actual provider typing system, and it is repeated three times within the higher-level grouping.
